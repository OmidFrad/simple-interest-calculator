# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful and inclusive experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Accepting constructive feedback
- Focusing on what is best for the community

Examples of unacceptable behavior include:

- Harassment, intimidation, or discrimination
- Insulting or derogatory comments
- Publishing others' private information without permission
- Any conduct that would reasonably be considered inappropriate in a professional setting

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope

This Code of Conduct applies within all project spaces and also applies when an individual is representing the project in public spaces.

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers. All complaints will be reviewed and investigated fairly.

## Attribution

This Code of Conduct is adapted for a basic GitHub training project.
